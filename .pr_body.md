- **Rebrands UI strings** "SCADA" → "MML Portal" across 5 frontend files (titles, sidebar logo, login hero, document.title suffix). No behavior changes — strings only.

## Why

`DEVELOPMENT.md` was written when login was the only auth feature. The system has grown significantly since then (HttpOnly cookies, admin user management, password reset flow, SMTP via Brevo) and the doc was misleading — e.g. it claimed the refresh token lived in `localStorage` and that the interceptor would clobber wrong-password errors, both of which are no longer true. There was also no top-level `README.md`, so newcomers had no deployment-focused entry point.
