1 Browser Request

The browser navigates to the MML Portal URL. Vite dev server (or Nginx in production) receives the request and serves index.html. This is a standard Single Page Application shell — a nearly empty HTML file with a <div id="app"> mount point and a single <script type="module" src="/src/main.js"> tag.
