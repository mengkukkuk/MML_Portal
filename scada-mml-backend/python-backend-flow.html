1 Uvicorn — ASGI Server Entry

When the Vue frontend sends an HTTP request (e.g. POST /api/auth/login), Uvicorn receives it on 0.0.0.0:8088. Uvicorn is an ASGI server — it handles async I/O and passes each request to the FastAPI app as an ASGI scope. In production you'd run it behind Nginx or NSSM (Windows service). The /health endpoint is available without auth for liveness monitoring.
